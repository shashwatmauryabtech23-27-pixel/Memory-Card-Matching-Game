# Memory Card Matching Game

## Overview

Memory Card Matching Game is a simple and interactive web-based game where players flip cards to find matching pairs. The goal is to match all pairs in the least number of moves and shortest time.

## Features

* Interactive card flipping animation
* Score tracking
* Move counter
* Timer functionality
* Restart game option
* Responsive design for desktop and mobile

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Play

1. Start the game by opening `index.html` in your browser.
2. Click on a card to flip it.
3. Flip another card to find its matching pair.
4. If both cards match, they remain open.
5. If they do not match, they flip back after a short delay.
6. Continue until all pairs are matched.

## Installation

1. Clone this repository:
